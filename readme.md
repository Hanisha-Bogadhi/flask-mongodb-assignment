# Flask Frontend-Backend with MongoDB Atlas

## Project Overview
This assignment demonstrates a full-stack Flask application with:
- Frontend (HTML form using Flask)
- Backend API (Flask)
- MongoDB Atlas database integration
- JSON file handling

---

## Features

### 1. API Endpoint
- Route: `/api`
- Reads data from a JSON file (`data.json`)
- Returns JSON response

### 2. Form Submission
- User enters name and email
- Data is sent from frontend to backend
- Backend stores data in MongoDB Atlas

### 3. Success & Error Handling
- On success → redirects to success page
- On failure → shows error on same page

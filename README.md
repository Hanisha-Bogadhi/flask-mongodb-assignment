# Flask MongoDB Signup Application

## Overview
This is a full-stack Flask web application that allows users to sign up through a form, stores the data in MongoDB Atlas, and provides an API to retrieve stored data.

## Features
- User signup form (Frontend - Flask)
- Backend API using Flask
- MongoDB Atlas integration
- Data storage and retrieval
- REST API endpoint to view data
- Error handling and redirection
